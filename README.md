This project implements palmprint recognition using Python and computer vision techniques. It uses OpenCV, NumPy, and TensorFlow to extract and analyze features from palm images. Algorithms like SIFT are applied to detect distinctive patterns for biometric identification. The system compares different methods and generates visual outputs showing their performance.

To run the project, the palm dataset palmprint.zip must be available in the same Google Drive environment. Without the dataset, the code cannot process or compare images.
